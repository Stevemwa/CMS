## Installation Steps

### 1. Clone the repository and change directory into it
```
git clone https://github.com/kaburia/CMS.git
cd CMS
chmod +x install.sh
./install.sh
```


Open [http://localhost:5500](http://localhost:5500) with your browser to see the result.
